## 這是什麼？

[Succumate -サキュメイト- (與魅魔莉莉姆的同居生活)](https://store.steampowered.com/app/1278390/)
[あまえんぼ (夏日狂想曲：鄉間的難忘回憶)](https://store.steampowered.com/app/1227890/)
那就可以用這個工具把遊戲的圖片換成無碼的。
